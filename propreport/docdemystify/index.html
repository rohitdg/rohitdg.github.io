Deed

  • Used for :
    1. Finding the owner and how he got the ownership of the property
    2. Check how the property devolved on the owner
    3. Size, location and surroundings of property
    4. To find the corresponding document in goverment records -
      [ Document No. + Year + SRO]

Encumberance Certificate

  • Used for :
    1. Finding all the registered transactions on the property
    2. Check if there are any encumberances on the property
    3. Check if there are any transactions on the property after the owner who is trying to sell got ownership
    4. Check for any outstanding mortgages on the property

Certified Copies

  • Used for :
    1. Finding all the registered transactions on the property
    2. Check if there are any encumberances on the property
    3. Check if there are any transactions on the property after the owner who is trying to sell got ownership
    4. Check for any outstanding mortgages on the property

Patta

  • Used for :
    1. Finding the extent of the owner
    2. Establishing possesion/ownership of the property

CMDA Approval

  • Used for :
    1. Establishing that the construction plan has been authorized by CMDA
    2. Approval is also trust-fall for property clerance

Completion Certificate

  • Used for :
    1. Establising the constructed property was as per norms and the approved plan
    2. Safeguard from any future liabilities/demolition due to deviation from plan

Property Tax Receipt

  • Used for :
    1. Checking for any dues to be cleared
    2. Support proof for ownership/possesion

Latest Electricity Bill

  • Used for :
    1. Getting the consumer id - To check for any outstanding dues
    2. Support proof for ownership/possesion
